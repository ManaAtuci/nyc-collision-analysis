# NYC Collisions Analysis
Analysis of NYC motor vehicle collisions from 2020-2023."

## Project Overview
This project analyzes 238,000 motor vehicle collisions reported by the NYPD from 2020-2023. The goal was to identify patterns, key contributing factors, and high-risk areas to propose data-driven safety recommendations.

## Key Insights
*   **Total Accidents:** 238,000
*   **Total Fatalities:** 635
*   **Most Dangerous Street:** Belt Parkway (1.55% of all accidents).
*   **Most Common Cause of Fatalities:** Unsafe Speed.
*   **Highest-Risk Time:** Friday afternoons (3-6 PM).
